AmeriCommerce - Help Scout Custom App

A custom app to pull customer information into Help Scout converstaions.

Installation

  1. create an API key in your AmeriCommerce admin panel
  2. while logged into Help Scout, go to "Manage > Apps"
  3. scroll to bottom, click "Build a Custom App"
  4. click "Create App"
  5. name the app. ex: "Customer Info" or "AmeriCommerce"
  6. set Content Type to "Dynamic Content"
  7. set Callback Url to "https://achs.herokuapp.com/api/WWW.YOURSTORESSLDOMAIN.COM/YOURAPIKEY"
  8. set Security Key to any random string.
  9. select which mailboxes you want to display customer information in.
  10. SAVE.
